The update uses an ajax call to: ../posts.asmx/walkupdate.

This

  • Executes an SP: Update_PlotLocation in OnlineApplications on the Web Server which updates the field: GISOverride in PublicCemeteries.dbo.BurialRecords or Cemeteries.dbo.Plot depending on the first character (G or S) of the ID
  • Writes the image to: F:\inetpub\online\assets\cemetery\images\[G/S][plotid]\Draft\WalkUpdate_[counter].jpg
